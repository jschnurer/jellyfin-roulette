# Local settings
Create `local.settings.json` and match this schema:

```
{
    "jellyfin": {
        "apiUrl": "yourjellyfinurl:8096",
        "apiKey": "your api key",
        "userId": "your user id",
        "movieLibraryId": "your movie library id"
    }
}
```

# Running
Run npm start or `node .\index.js`.
